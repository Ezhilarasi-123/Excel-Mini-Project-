# Customer Lifetime Value & Churn Risk Analytics
# Project Objectives
This project aims to analyze customer churn behavior and Customer Lifetime Value (CLV) using the Telco Customer Churn dataset. The main objective is to identify customers who are at high risk of leaving, understand the factors influencing churn, and evaluate customer value for business decision-making. Using Excel dashboards and SQL queries, the project provides actionable insights that help improve customer retention, reduce churn, and increase long-term profitability.
An interactive data analytics project that analyzes Customer Lifetime Value (CLV) and customer churn using SQL and Excel dashboards to identify high-risk customers and support customer retention strategies.

 # Dataset Used
The project uses the Telco Customer Churn dataset, which contains information about 7,043 customers from a telecommunications company. The dataset includes customer demographics, internet service type, contract details, payment methods, monthly charges, customer lifetime value, tenure, paperless billing, technical support, online security, and churn status. This data enables comprehensive analysis of customer behavior, service usage, and churn patterns.


## 📂 Dataset

**Dataset:** [Telco Customer Churn Dataset (Kaggle)](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)


# Key Questions Answered

1.What is the overall customer churn rate and how many customers have churned?

2.Which customers belong to High, Medium, and Low Risk categories?

3.Which contract type, payment method, and internet service have the highest churn?

4.How do gender, senior citizens, paperless billing, and technical support influence churn?

5.Which customer segments generate the highest Customer Lifetime Value (CLV), and what strategies can improve customer retention?

# KPI Dashboard & Interaction
An interactive Excel dashboard was developed using Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, and SQL-generated insights. The dashboard displays important KPIs including Total Customers, Churn Customers, Churn Rate, Average CLV, and High-Risk Customers. Interactive slicers allow users to filter the dashboard by Gender, Contract Type, Customer Segment, Internet Service, Payment Method, Risk Level, Tech Support, Paperless Billing, Senior Citizen, and Churn Status. These filters enable dynamic analysis and help users quickly identify trends and customer behavior.

#  Project Process
1.Imported the Telco Customer Churn dataset into SQL and Excel.

2.Cleaned and prepared the data by handling missing values, correcting data types, and removing inconsistencies.

3.Used SQL queries to calculate KPIs such as Total Customers, Churn Rate, Average CLV, High-Risk Customers, and customer segmentation.

4.Imported the processed data into Excel and created Pivot Tables, Pivot Charts, KPI Cards, and Interactive Slicers.

5.Built an interactive dashboard to visualize churn patterns, customer lifetime value, and key business metrics for better decision-making.

# Project Insights
1.The dataset contains 7,043 customers, of which 1,869 customers have churned, resulting in a 26.5% churn rate.

2.826 customers were identified as High-Risk customers requiring immediate retention efforts.

3.Customers with Month-to-Month contracts showed significantly higher churn compared to One-Year and Two-Year contracts.

4.Customers using Fiber Optic internet service and Electronic Check payment methods experienced higher churn rates.

5.Customers without Technical Support and Online Security were more likely to leave the company, while higher Customer Lifetime Value (CLV) customers contributed the most to long-term business revenue.
